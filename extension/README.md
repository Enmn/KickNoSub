## Supported Browsers
This extension works on all **Chromium-based browsers**, including:
- Google Chrome
- Microsoft Edge
- Brave
- Opera

## Installation (Developer Mode)
1. Open your **Chromium-based browser** (Chrome, Edge, Brave, etc.)
2. Go to **Settings** → **Extensions** (or Manage extensions)
3. Enable **Developer mode** (top-right corner)
4. Click **Load unpacked**
5. Select the `extension/` folder from this repository

Then open a Kick VOD page, And it will work!

## Notes
- It is currently in **Beta Version**.
- Functionality depends on Kick’s site structure.
